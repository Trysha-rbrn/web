## Опис

Промените да ли је играч у модусу посматрача или не. Док је у модусу посматрача, играч може посматрати (гледати) друге играче и возила. Након коришћења ове функције, потребно је користити или [PlayerSpectatePlayer](PlayerSpectatePlayer) или [PlayerSpectateVehicle](PlayerSpectateVehicle).

| Назив        | Опис                                       |
| ----------- | ------------------------------------------------- |
| playerid    |ID играча који треба да буде у модусу посматрача.          |
| bool:toggle | **true** за укључивање посматрања и **false** за искључивање. |

## Враћа

**true** - Функција је успешно извршена.

**false** - Функција није извршена. Играч не постоји.

## Примери

```c
public OnPlayerDeath(playerid, killerid, WEAPON:reason)
{
    TogglePlayerSpectating(playerid, true);
    PlayerSpectatePlayer(playerid, killerid);
    return 1;
}
```

## Белешке

:::tip

Када се модус посматрача искључи, функција OnPlayerSpawn ће бити аутоматски позвана. Ако желите да вратите играча у стање пре посматрања, морате то обрадити у OnPlayerSpawn. Такође, играч може прећи на избор класе ако је користио F4 током посматрања, а играч такође МОЖЕ умрети у овом режиму због различитих гљичева.

:::

:::tip

Када је играч у режиму посматрача, његов HUD је сакривен, што је корисно за подешавање камере играча без HUD-а. Такође, објекти близу камере играча ће бити учитани, што је корисно за интерполацију камера.

:::

:::warning

Ако играч није учитан пре него што се статус посматрања подеси на false, веза може бити неочекивано прекинута.

:::

## Повезане функције

- [PlayerSpectatePlayer](PlayerSpectatePlayer): Посматрајте другог играча.
- [PlayerSpectateVehicle](PlayerSpectateVehicle): Посматрајте возило.
